# Install
Requires NodeJs latest version, GIT and ```npm install -g npm-check-updates grunt-cli npm```

## Windows / Powershell
```powershell
iwr https://raw.githubusercontent.com/patrikx3/corifeus/master/scripts/init.ps1 -UseBasicParsing | iex
```

<!--
# or, add credit
